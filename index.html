Single cell analysis - Clustree

Ashwani Kumar

Rationale

Single cell analysis enables us to decipher cellular heterogeneity at the cellular level. Most of the tools employed for analysis of single cell expression data uses dimension reduction, finding neighbors and cluster cells into similar cell types.However how do we make sure that all the cells in a cluster represent a specific cell type. We always see expression of markers specific to celltype appearing in a single cell cluster. How do we deal with such situations.

The above described question is a function of 3 inputs. A) The method of clustering B) The Similarity function used in clustering. C). The similarity threshold.In tools like Seurat (https://satijalab.org/seurat/), while find cluster we used an argument called resolution which decides granularity of the clustering. Increasing or decreasing the granularity (similarity threshold) helps to capture either minor or coarse details of functionality of clustered cell types.

Using clustree: (https://github.com/lazappi/clustree),we can visualize and investigate clusters at different resolution.
