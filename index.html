Data normalization using scTransform

scTransform

Reference: Hafemeister, C., Satija, R. Normalization and variance stabilization of single-cell RNA-seq data using regularized negative binomial regression. Genome Biol 20, 296 (2019). https://doi.org/10.1186/s13059-019-1874-1

Performs negative binomial regularized normalization of gene expression data.

Rationale

  1. Per gene glm model parameters are learned. This can be done on a subset of genes/cells to make the run faster.

  2. sctransform gives user an option to choose methods. For example, if set to ‘poisson’ regression is done; negative binomial theta parameter is estimated using the response residual. If method is set to ‘glmGamPo’, coefficient and theta are estimated by a call to glmGamPoi::glm_gp.

  3. Regularize parameters, both offset and slope.

  4. Transform UMI counts into person residuals.

Variance stabilizing transformation(seurat::vst)

  1. Uses a regularized negative binomial regression model

  2. Returns corrected UMI matrix counts and pearson residual

  3. Regress out the unwanted effects of UMI counts

    Usage :

    vst( umi, cell_attr = NULL, latent_var = c(“log_umi”), batch_var = NULL, latent_var_nonreg = NULL, n_genes = 2000, n_cells = NULL, method = “poisson”, do_regularize = TRUE, theta_regularization = “od_factor”, res_clip_range = c(-sqrt(ncol(umi)), sqrt(ncol(umi))), bin_size = 500, min_cells = 5, residual_type = “pearson”, vst 23 return_cell_attr = FALSE, return_gene_attr = TRUE, return_corrected_umi = FALSE, min_variance = -Inf, bw_adjust = 3, gmean_eps = 1, theta_estimation_fun = “theta.ml”, theta_given = NULL, exclude_poisson = FALSE, use_geometric_mean = TRUE, use_geometric_mean_offset = FALSE, fix_intercept = FALSE, fix_slope = FALSE, scale_factor = NA, vst.flavor = NULL, verbosity = 2, verbose = NULL, show_progress = NULL )

  4. Returns: (please refer to sctransform: Variance Stabilizing Transformations for Single Cell UMI Data (r-project.org)

    umi_corrected: Matrix of corrected UMI counts

    model_str : Representation of model formula

    model_pars: Matrix of estimated model parameters per gene

    cell_attr: Data frame of cell meta data

    gene_attr: Data frame of gene attributes

    #Apply VST 
x10_vst <- sctransform::vst(x10.df,latent_var = c("log_umi"),return_cell_attr=TRUE, return_gene_attr=TRUE, return_corrected_umi=TRUE, verbosity = 1)
